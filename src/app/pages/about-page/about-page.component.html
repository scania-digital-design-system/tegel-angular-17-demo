About this page

This page is a testing ground for the variable migration in @scania/tegel-angular-17@1.52.0-var-batch-4-beta.0 and @scania/tegel-lite@0.1.0-var-batch-4-beta.0. Each section shows the Stencil web components (tds-*) first, then the Tegel-Lite CSS-only equivalent (tl-*). Use the Brand toggle in the top-right to switch between Scania and Traton.
