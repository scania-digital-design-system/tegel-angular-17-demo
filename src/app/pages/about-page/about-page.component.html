About this page

This page is a testing ground for the variable migration in @scania/tegel-angular-17@1.51.0-var-work-beta.1. It showcases the affected components (Link, Breadcrumbs, Divider, Footer) in multiple states so every edge case is testable. Use the Brand toggle in the top-right corner to switch between the Scania and Traton brands.
